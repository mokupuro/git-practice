有名どころの言語の特色

  • python

    • 今かなり人気の高い言語。主にデータ分析、web系、AI、人工知能、機械学習等に使われている。

  • java script

    • 古くから人気の高い言語であり、様々な学習サイトで学ぶことが出来る。
    主にフロントエンド、ゲームのデザイン、webページ作成、ゲーム作成等に使われている。

  • java
    • 引用元：https://thiscalifornianlife.com/2018/12/18/java11-01/

    基本的に何でもできる言語。javaで作られたゲームには、かの有名なマインクラフトがある。
    java scriptと名前は似ているが全くの別物である。

  • Ruby

    • 出来ることはpythonとほぼ一緒。しかし日本人が作った言語なので学びやすいという利点がある。
    知らぬ者がいないTwitterやクックパッドもRubyで作られた。

  • C言語
    • 引用元：https://www.bizlearn.jp/bizlearn-tech/530

    こちらも何でもできる言語。しかし作られたのがあまりにも古いためネットワークに弱い。
    またjava等では自動化されているものも全て自分の手で書かなければならない。 極めれば高収入を期待できる。

  • HTML&CSS
    • 引用元：https://style.potepan.com/articles/22186.html

    webページ作成には欠かせない言語。このホームページもHTMLで作られている。
    初学者に向いている言語のため多くの人間が学ぶことになるだろうが、意外と極めるのは難しい。

    何の言語から始めるべき？

    実は何の言語から始めるかはあまり重要ではない。
    初学者向けとされる言語もあるが、基本的には自分の目的に合った言語を選択するといいだろう。
    一つの言語を習得すれば後は翻訳作業のようなものなので、とりあえず自分の目的に合った一つの言語を極めることが大事だ。

    オススメの言語の難易度

    早く極めてお金を稼ぎたいならpython

    (人気急上昇中の言語なので案件も増えてきている)

    頑張るならjava

    （難しい、でも極めたらずっと稼げるかも）

    本気ならC言語
